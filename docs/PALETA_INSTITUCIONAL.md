## Tipografia y logotipo

El texto extraido automaticamente no muestra una recomendacion tipografica explicita ni reglas de uso de logotipo. La interfaz usa una pila tipografica de sistema por legibilidad y disponibilidad.

## Uso en la aplicacion

Los tokens CSS estan definidos en `src/styles/tokens.css`. El azul profundo se usa como color estructural, el amarillo como acento, y los colores de estado se asignan con contraste suficiente para lectura.
